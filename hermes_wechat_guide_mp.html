Hermes 超级大脑

24h 私人 AI 微信助理部署指南

◆ 第一步：获取“灵魂” (Google API Key)

Hermes 的强大核心源自谷歌 Gemini 模型。你需要先申请一把免费的“钥匙”。

  1. 访问 Google AI Studio 官网（复制到浏览器打开）：
    aistudio.google.com
  2. 登录你的 Google 账号，点击左侧蓝色的 “Get API key” 按钮。
  3. 点击 “Create API key in new project”。
  4. 关键：复制那串以 AIza... 开头的字符并保存好。千万不要泄露给别人！

◆ 第二步：在电脑 (WSL) 中植入程序

在你的 Ubuntu (WSL) 终端中，执行官方提供的一键安装脚本。它会自动配置 Python 和 Node.js 环境。

curl -sL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

安装完成后，刷新系统环境：

source ~/.bashrc

◆ 第三步：精准驯服（核心设置）

输入 hermes setup 开始配置。请注意以下核心选择点：

1. 智商选择：遇到 Model 名称输入时，手动填入：gemini-2.5-flash

2. 通道选择：挪到 Weixin 选项，按空格键出现 [✓] 再回车。

3. 安全策略：为了绝对防封，目前仅支持 私聊 (Direct Messages)。扫码后直接开启所有私聊权限。

◆ 第四步：启动微信联动

最后，输入指令启动网关：

hermes gateway

拿出手机，扫描终端里弹出的二维码。看到 “微信连接成功” 之后，你的私人 AI 助手就正式入驻微信了！

◆ 第五步：日常极客指令手册

  • 微信站岗：hermes gateway（此窗口保持运行）
  • 电脑直聊：新开窗口输入 hermes 即可本地操作
  • 修改设置：hermes setup gateway（可随时重扫码）
  • 官方文档：hermes-agent.nousresearch.com

◆ 为什么只开私聊？

腾讯对 @im.bot 官方号有严格限制。目前的配置采用 100% 官方正规通道，彻底杜绝了破解协议导致的封号风险。你可以通过将“机器人名片”推荐给好友的方式，让他们体验你的 AI 助理，既安全又优雅。
