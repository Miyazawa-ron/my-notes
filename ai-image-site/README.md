# AI图片生成中转站 — 搭建任务清单

> 目标：为国内朋友搭建一个自助AI图片生成网站，支持 GPT Image 2 / Seedance 2，手动充值余额管理。

---

## 🗂️ 第一阶段：账号与环境准备

- [ ] **OpenAI 账号** — 确认已有并充值余额（日本网络可直连）
- [ ] **Seedance 2 API** — 注册字节跳动火山引擎账号，申请 Seedance API 权限
- [ ] **域名** — 准备一个 .com 域名（推荐 Cloudflare 购买）
- [ ] **服务器** — 选择以下方案之一：
  - 简单方案：Cloudflare Pages（静态前端）+ Cloudflare Workers（后端逻辑）
  - 完整方案：VPS（推荐 Vultr 日本节点，约 $6/月）

---

## 🗂️ 第二阶段：技术环境准备

- [ ] **本地安装 Node.js** — 在 Windows PC 上装好
- [ ] **代码编辑器** — VS Code（已有）
- [ ] **GitHub 账号** — 已有（`miyazawa-ron`），用于代码托管和部署

---

## 🗂️ 第三阶段：确认需求细节（开始写代码前回答）

- [ ] 用户登录方式 — 注册/登录系统，还是只给朋友一个专属密码？
- [ ] 计费单位 — 按次收费（每张图固定价），还是充值余额扣费？
- [ ] 支持模型 — 只要 GPT Image 2，还是同时要 Seedance 2？
- [ ] 界面语言 — 中文界面？

---

## 🗂️ 第四阶段：搭建（逐步实现）

- [ ] 搭后端 API 服务（接收提示词 → 调用图片API → 返回结果）
- [ ] 搭前端页面（提示词输入、模型选择、图片展示）
- [ ] 做用户余额系统（数据库存余额，手动充值后台）
- [ ] 部署上线（绑定域名，公网可访问）
- [ ] 测试全流程

---

## 💡 支付方案说明

| 阶段 | 方案 | 说明 |
|------|------|------|
| 起步 | 个人收款码 + 手动充值 | 朋友扫码付款 → 截图给你 → 你手动加余额，零技术门槛 |
| 扩大 | 聚合免签支付（码支付/Epay等） | 自动回调充值，无需商户号，适合用量增大后升级 |
